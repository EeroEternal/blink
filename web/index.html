Blink

Agent 隔离沙箱执行面 · 供 XEnsemble 等控制面调用

检查中…

定位

Blink 不是用户控制台。用户登录、配额、Agent 管理由 XEnsemble 控制面负责；Blink 只提供 libkrun VM 沙箱的生命周期与执行 API。

集成文档

控制面开发者请参阅仓库内 docs/XENSEMBLE.md，了解 Runtime Provider 与 Blink API 的映射关系。

API

  • GET /api/health
  • GET /api/product
  • POST /api/runs — 单次执行
  • GET/POST /api/sessions — 持久 Session
  • POST /api/sessions/{name}/checkpoints — Snapshot

无 API 鉴权 · 默认 127.0.0.1:8787 · 鉴权由 XEnsemble 控制面负责
